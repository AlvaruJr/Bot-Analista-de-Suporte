# Bot Especialista - Projeto Conecta RS (Claro/SEDUC)

## Objetivo
Bot especialista desenvolvido para suporte técnico em campo e inventário de rede, utilizando inteligência artificial para interpretar logs reais de ativação e imagens de etiquetas de equipamentos.

## Funcionalidades
* **Suporte Técnico:** Baseado em logs reais do WhatsApp (procedimentos de IP, Reset e P1).
* **Inventário por Imagem:** Extração de MAC, Serial e Nome de APs, Switches e Firewalls.
* **Regra 55AXE:** Identificação automática de modelos Outdoor conforme padrão do projeto.

## Como Executar
1. Clone o repositório: `git clone <seu-link>`
2. Instale as dependências: `pip install -r requirements.txt`
3. Crie um arquivo `.env` com sua chave: `API_KEY_IA202601=SUA_CHAVE`
4. Execute: `python main.py`
